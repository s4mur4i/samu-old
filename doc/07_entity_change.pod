=head2 VM_change_function

=head3 SYNOPSIS

samu.pl vm change [options] [function]

=head3 OPTIONS

=over

=back

=head3 FUNCTIONS

=over

=item cdrom

Change the backend iso to a cdrom drive

=item interface

Change the network connected to a network

=item altername

Change the alternative name of a vm

=item snapshot

Revert to snapshot attached to a vm

=item power

Change the power state of a vm

=item disk

Change the size of harddrive

=back

=head2 VM_change_cdrom_function

=head3 SYNOPSIS

samu.pl vm change cdrom [options]

=head3 OPTIONS

=over

=item vmname

Name of vm

=item num

Number of cdrom

=item iso

VMware path to iso

=item unmoun

Unmount the attached iso

=back
